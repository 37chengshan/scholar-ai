# ScholarAI 前端 UAT 测试计划

## 测试环境
- 前端地址：http://localhost:3000

## 路由说明

**当前有效路由** (per routes.tsx):
- `/` - Landing Page
- `/login` - 登录页面
- `/register` - 注册页面
- `/forgot-password` - 忘记密码
- `/reset-password` - 重置密码
- `/dashboard` - 仪表盘
- `/knowledge-bases` - 知识库列表
- `/knowledge-bases/:id` - 知识库详情
- `/search` - 搜索页面
- `/read/:id` - 论文阅读页面
- `/chat` - AI问答页面
- `/settings` - 设置页面
- `/notes` - 笔记页面

**已移除路由**:
- `/upload` - 功能已整合到知识库导入流程 (KnowledgeBase Detail → Import)
- `/library` - 功能已整合到知识库 (KnowledgeBase)

## 测试场景

### 1. 登录页面测试 (LoginPage)
**测试目标**: 验证用户登录功能

**测试步骤**:
1. 访问 http://localhost:3000/login
2. 验证页面元素加载（邮箱输入框、密码输入框、登录按钮）
3. 输入测试账号：test@example.com
4. 输入测试密码：Test123456
5. 点击登录按钮
6. 验证是否跳转到首页

**预期结果**:
- 页面正常加载
- 登录成功后跳转到首页
- 登录失败显示错误提示
